Usage

Import/Export
Data may be loaded into the system and exported from it using xlsx. The simplest way to upload data is to download the excel file export and use it as a base for creating a file to upload.
Setup
To setup Sales Estimates information, click Process and add extra information under the "Setup" tab. The models should be self explanitory, faster editting/adding data may be possible by using teh "Mass Edit" button under each model.
Generating Results
Once All information has been endered under the "Setup" tab, you can swap to the generate tab to calculate Sales Estimates.
Results
Results may be viewed under the Results tab or downloaded through the export button.
